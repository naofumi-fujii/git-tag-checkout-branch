## tag-checkout-branch

## usage
add following line to `.gitconfig`
```
+  tag-checkout-by-branch = !git tag | peco | tag-checkout-branch
```
## peco (filter tool)
peco/peco: Simplistic interactive filtering tool https://github.com/peco/peco
